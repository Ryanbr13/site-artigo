O Basico sobre DNS: o que é? para que serve?

O DNS Domain Name System ou Sistema de Nomes de Domínio é a "lista telefônica" da internet. Ele traduz os nomes de domínio legíveis por humanos como google.com em endereços IP numéricos como 172.217.160.142 que os computadores usam para se comunicar.

  • Porque é importante

    • Permite que os usuários acessem sites usando nomes de domínio em vez de endereços IP numéricos.
